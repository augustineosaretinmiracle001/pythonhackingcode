# Frontend Application

Standalone frontend that connects to Python API backend.

## Setup

1. Update API URL in `script.js`:
   ```javascript
   const API_URL = 'https://your-deployed-api.com';
   ```

2. Open `index.html` in browser or deploy to hosting platform

## Deploy Options

### Netlify (Free)
1. Push to GitHub
2. Connect repository to Netlify
3. Auto-deploys on every push

### Vercel (Free)
1. Push to GitHub
2. Connect repository to Vercel
3. Auto-deploys on every push

### GitHub Pages (Free)
1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Select source branch

## Features

- Pure HTML/CSS/JavaScript
- Connects to separate Python API
- Responsive design
- Error handling
- API status checking
